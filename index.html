    Etiquetas de texto semanticas estructurales

    header representa la cabecera de una pagina web o seccion de contenido.

    nav se usa para representar la parte de navegacion de un sitio web o lo que se realice.

    Ejemplo de navegacion:

    • Inicio
    • otra cosa
    • Queso
    main se usa para agrupar el contenido principal de una pagina web.

    Section es para indicar en que parte debe ir cada contenido en un sitio web

    footer es el pie de la pagina, donde se muestran cosas como el Copyrigth, contactos, cosas legales y demas.

    article representa un articulo, como una publicacion de un blog, una notica, etc.

    aside representa los contenidos relacionados o no con el contenido principal, ejemplos: anuncios.

    address es donde iria la informacion de contacto, emails, numeros, direcciones, etc.

    Metaetiquetas para SEO y moviles

    Para esto se debe ir al head de nuestro documento HTML en nuestro editor

    link rel="canonica" href="https://tudominio.com/la-url-canónica" (Esta seria la URL canonica (significa que es la URL original) de nuestra pagina web)
    link rel="icon" href="multimedia/imagenes,etc/favicon.png" (El icono pequeño que aparece en la ventana del computador)
    link rel="apple-touch-icon" href="multimedia/imagenes,etc/favicon.png" (El icono pero para dispositivos moviles, en esta la resolucion debe ser la mejor)
    meta name="theme-color" content="#008000" para moviles (Para que en los moviles se vea en color la parte donde esta la URL, etc.)
    meta name="keywords" content="Practicar html, html, ejemplos"(Para poner palabras clave)
    meta name="author" content="Yo, Franco Mazzei" (Para poner el nombre del autor/a o autores)
    meta name="Copyrigth" content="empresa.inc" (Para especificar los derechos de autor)
    meta name="robots" content="noindex" (Para que los motores de busqueda sigan esos enlaces)
    meta name="robots" content="nofollow" (para que los motores de busqueda no sigan esos enlaces)

    Estas serian algunas de las metaetiquetas para SEO y dispositivos moviles, porque como se ve en el word, hay mas etiquetas que puede ser title o description que se encuentran en head donde estan todas las meta etiquetas. Esto es solo un ejemplo.

    ⬆️

    Meta etiquetas para redes sociales

    meta property="og:title" content="practikeishon" (Para que se vea el titulo de lo que compartamos en redes sociales)
    meta property="og:description" content="Tamos viendo html aca perreque" (Para que se vea la descripcion de lo que compartamos en redes sociales)
    meta property="og:image" content="https://tudominio.com/la-ruta-de-la-imagen" (Para poner la imagen referente que se vera al compartir la URL)
    meta property="og:url" content="https://tudominio.com/index.html" (Para poner la URL canónica o original y primero necesitamos tener un dominio)
    meta name="twitter:card" content="summary"> (El formato especial que tiene Twitter. twitter:card es para la apariencia que tendra la url al compartir y summary significa la descripcion breve que da)
    meta name="twitter:site" content="cuentadetwiter" (El formato especial de Twitter. twitter:site es para especificar el nombre del usuario y content es para poner el nombre del usuario)

    Mas información

    role, es un atributo que se usa para especificar a las herramientas de ayuda el rol semantico (significado) que cumple el elemento en la pag.
    role="title" indica que el rol es de un titulo.
    role="list" inidica que el rol es el de una lista.
    role="listitem" indica que un elemento HTML forma parte de una lista.
    tabiindex, este atributo se usa para poder navegar por la pagina con la tecla TAB del teclado, en caso de no tener mouse el usuario o que se le haya quedado sin bateria, etc.
    aria-label, este atributo se usa para dar una descripcion del elemento interactivo, como por ejemplo un link, daria una descripcion de lo que se vera en el link. Descripcion previamente puesta alli por la persona que haya hecho el documento HTML.
